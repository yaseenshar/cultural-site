# Pinned Repository About Content

## Short version (GitHub pinned card)
Full-stack cultural discovery platform built with Angular + Kotlin/Spring Boot. Includes JWT auth, role-based access, site catalog APIs, favourites workflow, and map-based exploration backed by PostgreSQL.

## Medium version (profile/project showcase)
Cultural Site Explorer is a production-style full-stack application that helps users discover and manage cultural places such as museums, theatres, restaurants, and artwork points. The frontend is built with Angular 18 and integrates map-based exploration, while the backend uses Kotlin + Spring Boot with JWT security, role-based authorization, and PostgreSQL persistence.

### Highlights
- End-to-end authentication (signup/login/refresh)
- Admin and user role separation
- Cultural site listing, grouping, and filtering
- Favourites management and profile flows
- Cleanly layered backend architecture (controller/service/repository)

### Stack
Angular 18 · TypeScript · Kotlin · Spring Boot · Spring Security · PostgreSQL · JPA · Tailwind CSS
